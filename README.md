# Java 训练营代码

## Maven 编译和运行

* 安装 JDK 17以上
* 安装 Apache Maven 3.8 以上
* 项目使用 `mvn compile` 编译
* 使用 `mvn exec:java -Dexec.mainClass="geektime.nio.CharSetDemo"` 执行测试
* `Dexec.args='XXX'` 提供运行参数
